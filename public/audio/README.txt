[영어 졸업 시험 음성 파일 안내]

영어 졸업 시험(영졸) 문제는 오디오 재생 기능이 있습니다.
이 폴더(public/audio/)에 음성 파일을 아래 규칙으로 넣어주세요.

파일명 규칙:
  part{파트번호}_set{세트번호}_q{문제번호}.mp3

예시:
  part2_set1_q1.mp3   → Part 2, Set 1, 1번 문제
  part2_set1_q2.mp3   → Part 2, Set 1, 2번 문제
  part2_set2_q1.mp3   → Part 2, Set 2, 1번 문제

* 파일이 없으면 플레이어는 표시되지만 재생되지 않습니다.
* 성경/컴퓨터 졸업 시험는 오디오 기능이 없습니다.
